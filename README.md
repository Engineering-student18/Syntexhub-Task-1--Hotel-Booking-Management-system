# Syntexhub-Task-1--Hotel-Booking-Management-system
The Hotel Booking Management System is a Java command-line application that simulates hotel room reservation operations. It allows users to view room availability, book rooms, cancel bookings, and handles invalid inputs efficiently. 

# Features
View available rooms
Book a hotel room
Cancel room bookings
Uses arrays to manage room data

# Technologies Used
Java
IntelliJ IDEA
